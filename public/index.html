如何让 Twitter 识别卡片链接
Twitter 的爬虫需要能从公网访问你的服务器。localhost 无法使用。推荐方案：
  1. ngrok（最快，适合测试） — 安装后运行 ngrok http 3000，得到形如 https://xxxx.ngrok-free.app 的公网地址。将该域名设为环境变量后重启服务：
    BASE_URL=https://xxxx.ngrok-free.app node index.js
  2. Railway / Render / Fly.io（永久部署） — 将本项目推到 GitHub，一键部署即可获得固定域名，设置 BASE_URL 环境变量为该域名。

生成后的卡片链接格式为 https://你的域名/c/<id>，粘贴进推文正文即可让 Twitter 自动展示图片卡。 如需验证 meta 标签是否正确，可用 Twitter Card Validator。
